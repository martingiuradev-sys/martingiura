# Logos

Por ahora la barra de credibilidad ("Una trayectoria construida en…") usa los nombres
en tipografía serif, que se ven muy prolijos. Si querés usar los logos reales, dejá
los archivos acá (preferentemente `.svg` monocromo, o `.png` con fondo transparente):

- `woow.svg`
- `piio.svg`
- `angelpad.svg`
- `forum-ventures.svg`
- `inter-miami.svg`
- `prospecta.svg`

Pasámelos (o dejalos en esta carpeta) y los cableo en `components/ProofBar.tsx`.
