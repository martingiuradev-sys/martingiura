# Imágenes

Colocá tu foto profesional acá con el nombre exacto:

- `martin-giura.jpg` — retrato vertical (idealmente ~1000×1250 px, formato 4:5).

Mientras el archivo no exista, el sitio muestra un monograma "MG" elegante en su lugar
(no se rompe el layout). Apenas agregues el `.jpg`, aparece automáticamente.

Formatos aceptados: `.jpg`, `.png`, `.webp` (si usás otro formato, actualizá la ruta
en `components/Portrait.tsx`).
